# MC-Server-Downloader
Download the newest spigot.jar with Multilanguage System (EN &amp; DE)

## Info

You need Java 8 on your Server.

## Installation

```bash
https://github.com/Realtox/MC-Server-Downloader.git
```
Go to the directory
Use 
```bash
chmod 777 MC-Installer.sh
```
And run it with 
```bash
./MC-Installer.sh
```
